# 超简便js代码实现“倒计时”

css样式省略

<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<title>时钟</title>
		<script type="text/javascript">
			function displaytime() {
				var djshi = document.getElementById("djs");
				djshi.innerHTML--;
				if(djshi.innerHTML<=0)
				{
					clearInterval(timeid);
				}
			}
		</script>
	</head>
	<body>
		<div id="djs">5
		</div>
		<script type="text/javascript">
			var timeid = setInterval("displaytime()",1000);
			
		</script>
	</body>
</html>

效果呈现就是5、4、3、2、1、0
